🌶️ Panch Phoron – Bengali Food Blogging Website
Panch Phoron is a simple and visually engaging food blogging website showcasing authentic Bengali recipes. Built using HTML, CSS, and JavaScript, this project focuses on clean UI, smooth navigation, and a pleasant browsing experience for food enthusiasts.

⭐ Features
🍽️ Authentic Bengali Recipes — A curated collection of traditional Bengali dishes
🎨 Responsive Design — Clean and mobile-friendly UI built with HTML & CSS
⚡ Interactive Navigation — JavaScript-based interactions for smooth module switching
📸 Recipe Sections — Categorized recipes for easy browsing
🍛 User-Friendly Layout — Minimal and intuitive interface

🛠️ Tech Stack
HTML5 – Page structure
CSS3 – Styling, layout, responsiveness
JavaScript – Interactivity & module navigation

🚀 How to Run the Project
Clone or download the repository
Open catalog.html or about.html in any browser
Navigate through recipes using internal links
No installation or setup required.

🎯 Objective
To create a simple, visually pleasing, and easy-to-navigate website showcasing traditional Bengali dishes and providing recipe details in a structured format.
